# Cinnamon
Discord bot built with Discord.js and Commando

## Requirements
NodeJS >=12.0.0
npm >=6.0.0

## Install
    $ git clone https://github.com/IncognitoCactus/cinnamon-bot
    $ npm i
    $ node cinnamon.js

## Configuration
If you would like to run Cinnamon on a local machine, you'll need a `secure.json` file with the following format:
`{
    "discordAPIKey": "[bot token here]"
}`

You can retrieve a bot token by going to https://discordapp.com/developers and creating a bot account.
